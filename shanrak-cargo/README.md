# SHANRAK CARGO - Сайт карго-компании

Одностраничный сайт для компании SHANRAK CARGO - доставка товаров из Китая в Астану и по всему Казахстану.

## Структура проекта

```
shanrak-cargo/
├── index.html      # Основной HTML файл
├── styles.css      # Стили CSS
├── script.js       # JavaScript для интерактивности
├── image.jpg       # Фоновое изображение для hero секции (нужно добавить)
└── favicon.ico     # Иконка сайта (нужно добавить)
```

## Особенности

- ✅ Полностью адаптивный дизайн (мобильные, планшеты, десктопы)
- ✅ Плавная прокрутка по якорям
- ✅ Анимации при скролле
- ✅ Оранжево-бело-черная цветовая схема
- ✅ Современный дизайн в стиле investinkids.kz
- ✅ SEO оптимизация
- ✅ Интеграция с WhatsApp, Instagram, 2GIS

## Секции сайта

1. **Hero** - Главный экран с фоновым изображением и призывом к действию
2. **О нас** - Информация о компании
3. **Услуги** - 4 карточки с услугами компании
4. **Отследить груз** - Ссылка на сервис отслеживания
5. **Отзывы** - 6 отзывов клиентов + скриншоты из WhatsApp
6. **Контакты** - Контактная информация и карта 2GIS
7. **Футер** - Дополнительная информация и ссылки

## Установка

1. Поместите файл `image.jpg` в корень папки `shanrak-cargo/`
2. Добавьте `favicon.ico` в корень папки
3. Откройте `index.html` в браузере
